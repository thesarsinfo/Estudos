Aula31-Formulario-p1

A tag form representa uma coleção de elementos com varios campos. Não é recomendado um form com um filho form

Caracteristicas

Diplay: Block

Categoria: Fluxo

Pai(contido): Fluxo

Filho: FLuxo

Atributos:

action: URL com o arquivo que irá receber os dados enviados

autocomplete: Auto completar os campos ativados

entype: Codificação(tipo) dos dados enviados

method: Metodo de envio dos dados (get(dados são visiveis) ou post(dados será feito internamente não visivel))

name: nome do formulário

novalidate: Desabilita a validação pelo browser

target: Página de destino(action) será aberta

-----------------

A tag label

Display: Inline

Categoria: Fluxo e fraseado

Pai: Fraseado

Filho: Fraseado

-----------------

A tag input

Display: Inline

Categoria: Fluxo , fraseado e interativo

Pai: Fraseado

Filho: Vazio

Atributos:

Type,name
