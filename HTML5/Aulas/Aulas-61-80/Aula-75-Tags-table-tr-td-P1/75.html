Aula-75-Tags-table-tr-td

A tag table serve para criar a estrutura(container) da tabela. A tag tr (table row)não é filho de table e cria uma linha e para cada celula da linha usamos um td (table division) com seu conteudo especifico

Caracteristicas

Elemento Table

Display: Block

Categoria: Fluxo

Pai(contido): Fluxo

Filho:caption, tr, thead , tfoot
