Aula-60-Tag-Object

A tag object incorporar um arquivo externo ao html da pagina e param fica mais facil configurar os parametros de object usando php ou javascript

Caracteristicas

Display: inline

Categoria: Fluxo , fraseado, Incorporado

Pai(contido): Onde incoporado é esperado

Filho: param

Atributos: data , type , name, usermap, form , width e height
