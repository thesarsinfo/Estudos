Aula 18 - Tags - code - pre

A tag code serve para redenrenrizar um texto de codigo de programação mais bonito e visivel e a tag pre é de pré formatação com espaços e quebra de linha

A tag code tem o display inline da categoria fluxo e fraseado e contido fraseado e tem filhos fraseado

A tag pre tem o display block da categoria de fluxo e contido por fluxo e seus filhos são fraseados

COde vem depois de pre devido a suas categorias
